# Armadilha: data diferente de hoje

O botão de salvar fica bloqueado quando a data selecionada não é a data atual.

## Efeito

Isso evita registrar a chamada diária em uma data incorreta.

## Complemento

No cadastro de alunos, a exclusão remove o registro do site e da planilha, mas o número já atribuído não deve ser reaproveitado.
