# Como excluir um aluno

## Passos
1. Abrir a página de edição do aluno.
2. Conferir se é realmente o cadastro correto, porque a exclusão remove a linha da aba `Cadastro`.
3. Usar o botão **Excluir Aluno** no topo da página, ao lado de **Voltar**.
4. Confirmar a ação no alerta de segurança.
5. Aguardar o retorno para a tela principal.

## Observação
O backend usa a ação `deletealuno` e remove a linha localizada pelo identificador atual do aluno.
