# Como salvar a chamada

1. Selecionar a data correta.
2. Escolher a turma.
3. Marcar pelo menos 1 aluno como presente, ausente ou atrasado.
4. Preencher oferta, visitantes, bíblias e revistas quando necessário.
5. Salvar mesmo que ainda existam alunos sem marcação; o sistema aceita salvamento parcial.
