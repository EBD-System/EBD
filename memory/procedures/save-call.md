# Como salvar a chamada

1. Selecionar a data correta.
2. Escolher a turma.
3. Marcar os alunos com presença ou atraso; os ausentes podem ficar sem envio para a planilha.
4. Preencher oferta, visitantes, bíblias e revistas quando necessário.
5. Salvar apenas os registros de presença/atraso que precisam ir para o Google Sheets.
