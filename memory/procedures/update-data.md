# Como atualizar os dados

1. Recarregar a tela quando houver necessidade de sincronizar com a planilha.
2. Confirmar se a turma selecionada continua válida.
3. Reabrir a chamada correspondente à data escolhida.

## Cadastro de alunos

1. Use a aba **Adicionar** para criar novo aluno com número automático.
2. Use a aba **Editar / excluir** para alterar nome, celular ou turma sem editar o número após `#`.
3. Use o botão de exclusão para remover o aluno do cadastro quando necessário.
4. Se a edição via site falhar, o botão de WhatsApp continua disponível como fallback manual.
