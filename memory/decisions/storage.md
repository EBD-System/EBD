# Rascunhos locais

## Decisão

Rascunhos de chamada, cache de turma e memória consolidada ficam no navegador quando for necessário reter estado temporário.

## Motivo

Isso reduz perda de edição durante o preenchimento da chamada e não interfere no histórico definitivo salvo no backend.

## Complemento

Dados de cadastro de aluno não devem depender do navegador como fonte persistente; o backend em Apps Script continua sendo a referência oficial.
